## Usage

> Make sure that you have [installed dependencies](#installing-dependencies).
> From within the root directory, run:
```sh
npm start
```

## Requirements

- Node.js
- Express ^4.15.2
- Mongoose ^4.9.1
- React ^15.4.2
- React-DOM ^15.4.2

## Development

### Installing Dependencies

From within the root directory:
```sh
npm install

```
